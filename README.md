# 3HP App - Downloads

This repository hosts the official APK releases for the **3HP App**.

> ⚠️ This repo contains **release artifacts only**. The source code is in a private repository.

## 📥 Download Latest

Check the [Releases page](https://github.com/mayberedfy/3hp-app-releases/releases) for the latest APK.

## 📱 Installation

1. Download the latest `.apk` file from the Releases page
2. On your Android device, allow "Install from unknown sources"
3. Open the downloaded APK to install

## 🔐 Verification

Each release includes SHA-256 checksums to verify file integrity.
